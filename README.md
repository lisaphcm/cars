# cars
### Logs and Lists Cars

### Two Components:
* cars.sh script
	* program to add/list cars
* My_old_cars file
	* stores all cars

### How to Use?
cars.sh has three menu options:
1. Enter a Car
	* Press 1
	* Takes in the YEAR, MAKE, and MODEL from the User
	* Adds new car to My_old_cars
2. List all Cars
	* Press 2
	* Prints My_old_cars as a list 
	* Sorted by YEAR
	* Colons replaced by spaces
3. Quit Program
	* Press 3
